# Medical Appointment No-Show Analysis

## Objective
To analyze why patients miss medical appointments and identify patterns influencing no-shows.

## Contents
- `Medical_Appointment_No_Show.ipynb` — Data cleaning and EDA notebook
- `appointments.csv` — Original dataset
- `output.png` — Visualization of key insights

## Key Steps
1. Data cleaning and preprocessing
2. Exploratory data analysis
3. Feature engineering
4. Model evaluation

## Insights
- Older patients are more likely to attend.
- SMS reminders reduce no-shows slightly.
